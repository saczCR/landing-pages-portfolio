# Landing Pages Portfolio

Colección de páginas web (landing pages) de promoción y venta, desarrolladas como proyectos de práctica. Cada carpeta contiene un proyecto independiente con su propio README.md detallando tecnologías y características.

## Proyectos

| Proyecto | Descripción | Tecnologías |
|---|---|---|
| [car-detailing/](./car-detailing) | Landing page para un curso online de detailing automotriz, con slider interactivo "antes/después" | HTML5, CSS3, JavaScript |
| [ia-heroes/](./ia-heroes) | Landing page para un evento en vivo de IA aplicada, con timeline y FAQ nativo | HTML5, CSS3, SVG |

## Stack general

- HTML5 semántico
- CSS3 (variables, Grid, Flexbox, animaciones, diseño responsive)
- JavaScript vainilla (sin frameworks)
- Diseño accesible (aria-label, focus-visible, prefers-reduced-motion)
