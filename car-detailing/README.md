# Landing Page — Curso de Car Detailing Profesional

Página web de promoción/venta para un curso online de detailing automotriz. El objetivo del sitio es presentar el curso, generar confianza y llevar al usuario a la sección de precios.

## Demo

Abre index.html en tu navegador, o publícalo con GitHub Pages (ver sección abajo).

## Tecnologías

- HTML5 semántico
- CSS3: variables (custom properties), Grid y Flexbox, animaciones con @keyframes, diseño responsive con media queries
- JavaScript (vainilla): interactividad sin librerías

## Características principales

- Slider interactivo "antes / después": comparación visual mediante un input type="range" y clip-path, sin depender de ninguna librería externa.
- Acordeón de preguntas frecuentes (FAQ) con animación de apertura/cierre.
- Diseño responsive: se adapta a móvil, tablet y escritorio.
- Accesibilidad: uso de aria-label en controles interactivos y respeto a prefers-reduced-motion para usuarios sensibles a animaciones.
- Tipografía a medida: combinación de tres familias tipográficas (Oswald, Inter, IBM Plex Mono) para jerarquía visual clara.
